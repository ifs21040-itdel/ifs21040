Apa itu bahasa Pemograman?

Bahasa pemrograman adalah alat yang digunakan untuk menginstruksikan komputer dalam mengeksekusi tugas-tugas tertentu. Mereka adalah sarana bagi kita untuk berkomunikasi dengan mesin-mesin ini secara efektif. Berikut ini adalah beberapa konsep dasar yang perlu Anda ketahui ketika Anda ingin mengenal lebih dalam tentang bahasa pemrograman.. Ada beberapa bahasa pemrograman harus dikuasai oleh seorang front-end developer yaitu HTML,CSS, dana JavaScript.

  1. HTML

    HTML adalah singkatan dari "HyperText Markup language". Ini merupakan bahasa pemrograman yang digunakan untuk membuat halaman web dan menentukan struktur dari konten halaman web. HTML menggunakan markup(penanda) dalam bentuk tag untuk mengidentifikasi elemen-elemen seperti teks, gambar, tautan, formulir, dan banyak lagi didalam halaman web.

    Berikut beberapa konsep dasar:

    • Markup Language : Bahasa markup yang menggunakan tag pengelompokan dan pemformat konten dalam halaman web
    • Elemen : Bagian dari dokumen HTML
    • Tag : digunakan untuk menandai elemen-elemen dalam dokumen. Misalnya dalam format <tag>konten</tag>
