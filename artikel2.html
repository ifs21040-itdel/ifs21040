HTML Programming

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan memformat halaman web. HTML adalah dasar dari hampir setiap situs web di internet. Ini memungkinkan kita untuk membuat teks, gambar, tautan, dan elemen-elemen lainnya menjadi bagian dari sebuah halaman web.

Apa yang Dapat Anda Lakukan dengan HTML?

Dengan HTML, Anda dapat melakukan banyak hal, termasuk:

  • Membuat judul dan paragraf
  • Menambahkan gambar dan video
  • Membuat tautan ke halaman lain
  • Membuat formulir input
  • Menyusun teks dalam tabel

Syntax Dasar HTML

Syntax dasar HTML terdiri dari tag-tag (elemen) yang mengelilingi teks atau elemen-elemen lainnya. Setiap tag memiliki fungsi tertentu. Sebuah tag biasanya memiliki format seperti ini:

        <tag>konten</tag>

Contoh tag HTML yang umum digunakan:

  • <html>: Menandakan awal dan akhir dari dokumen HTML.
  • <head>: Berisi informasi tentang dokumen, seperti judul.
  • <title>: Menentukan judul halaman yang akan ditampilkan di bilah judul browser.
  • <body>: Berisi konten yang akan ditampilkan di halaman web.
  • <p>: Membuat paragraf teks.
  • <a>: Membuat tautan (link) ke halaman lain.
  • <img>: Menampilkan gambar di halaman.

Ini adalah dasar-dasar HTML yang dapat Anda gunakan untuk memulai pembuatan halaman web sederhana. Teruslah belajar dan eksperimen dengan HTML untuk membuat halaman web yang lebih kompleks.
